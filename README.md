# 🌐 Personal Portfolio

🚀 Live Demo: https://portfolio-lake-eta-29.vercel.app/

This is my personal portfolio website showcasing my skills, projects, and experience as a Frontend Developer. It is designed to present my work professionally and provide an easy way for recruiters and clients to connect with me.

## ✨ Features
- 📱 Fully Responsive Design (Mobile + Desktop)
- 🎨 Modern UI with Clean Layout
- 🌙 Dark & Light Mode Toggle
- ⚡ Smooth Animations & Interactions
- 📂 Project Showcase Section
- 📩 Contact Form (Web3Forms Integration)

## 🛠️ Tech Stack
- HTML5 – Structure
- Tailwind CSS – Styling
- JavaScript – Functionality
- Web3Forms – Contact form handling

# 📸 Sections Included
- 🏠 Home
- 👩‍💻 About Me
- 🛠️ Skills
- 💼 Projects
- 📬 Contact
- 🚀 Getting Started

To run this project locally:

## Clone the repository
git clone https://github.com/Sundari31/Portfolio.git

## Navigate to the folder
cd Portfolio

## Open index.html in browser
